## Sobre a Automação de Cadastro de Produtos

Automatizando cadastros que demandaria tempo de serem feitos manualmente e ajudando a reduzir tempo, por exemplo. Simples, mas eficaz.

## Como usar?

- Abra o executável de cadastros em 'setup'.
- Abra o arquivo 'app.py' e execute o código.
- Com a ajuda da library pyautogui, o próprio mouse faz automaticamente a tarefa de: clicar no campo > inserir usuário > clicar em cada campo do produto e inserir automaticamente as informações inseridas em 'produtos.txt'.
- O código identifica automaticamente a partir de um looping 'for' que identifica que a cada vírgula no arquivo 'produtos.txt' será uma informação diferente para cada campo do cadastro.
